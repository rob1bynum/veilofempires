# Veil of Empires Roll20 v2.4.0 This release begins the production architecture migration while preserving the complete v2.3.3 character sheet. ## Visible changes None. The banner, portrait, header, Agent page, Protocols, Equipment, Dossier, Rules, roll templates, and CSS remain unchanged. ## Internal changes - Added `VOE_DATA`, a structured facade over the existing Heritage, Specialty, Protocol, Condition, Attribute, and Skill data. - Added `getRuleRecord()` as a stable lookup boundary for later migrations. - Added hidden schema and build-version attributes. - Kept all v2.3.3 calculations authoritative to avoid character-data or rules regressions. ## Roll20 Sandbox installation Paste these files into the corresponding Sandbox fields: - `Veil of Empires/veil-of-empires.html` → HTML Layout - `Veil of Empires/veil-of-empires.css` → CSS Styling - `Veil of Empires/translation.json` → Translation The banner continues to use the repository-hosted image URL already present in v2.3.3.

# Veil of Empires Roll20 v2.4.4

This release preserves the complete v2.4.3 sheet and adds faster loadout management.

## New features

- Protocol dropdown selectors are now available directly on the Protocols tab.
- Changing a Protocol in either the Agent or Protocols tab updates the same loadout slots.
- Equipment tab includes an Agent Requisition Catalog pop-up.
- Weapon selections create ready-to-use repeating weapon rows.
- Weapon target values are copied from the Agent's current Close Combat or Ranged Combat total.
- Armor selections add an inventory row and equip the selected armor.
- Base Equipment selections add standard items to the repeating inventory.
- A Standard Agent Field Kit can be added with one button.
- Existing custom repeating Protocol, Weapon, and Equipment entry remains available.

## Roll20 Sandbox installation

- `Veil of Empires/veil-of-empires.html` → HTML Layout
- `Veil of Empires/veil-of-empires.css` → CSS Styling

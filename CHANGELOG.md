# Changelog

## v2.4.2 - Round Actions Restored

- Set every Skill base default to 15.
- Replaced Round Actions text counters with reliable read-only Roll20 input fields.
- Added an explicit Attacks Remaining counter.
- Close and Ranged attacks now decrement Actions Remaining and Attacks Remaining while incrementing Attacks Made.
- Move decrements Actions Remaining only.
- Reset Round restores 3 Actions, 3 Attacks Remaining, 0 Attacks Made, and the Reaction.
- Existing v2.4.0 visuals, artwork, fields, rules, rolls, and data facade are preserved.

# Version 3.0 Roadmap

## 3.0-alpha.1: Architecture foundation
- structured data schema
- pure modifier engine
- Roll20 adapter boundary
- wizard model
- Player, NPC, and Creature mode definitions
- validation suite

## 3.0-alpha.2: Legacy integration
- import complete v2.3.3 HTML, CSS, and worker
- connect Conditions to the new engine
- connect Warden and all existing Specialties
- parity fixtures for existing calculations

## 3.0-alpha.3: Full rules migration
- migrate all Heritages
- migrate all Protocols
- migrate armor and equipment
- centralize summaries and tooltips

## 3.0-alpha.4: Production UI shell
- redesigned masthead
- combat dashboard
- card-based sections
- accessible controls
- responsive wide and narrow layouts

## 3.0-beta.1: Character Creation Wizard
- guided seven-step creation
- validation and review
- automatic starter choices
- resume-in-progress support

## 3.0-beta.2: NPC and Creature sheets
- shared Conditions and roll engine
- compact actions
- threat and instinct blocks
- token-friendly resources

## 3.0-rc.1: Release candidate
- migration testing
- Roll20 sandbox testing
- localization pass
- repository metadata and documentation

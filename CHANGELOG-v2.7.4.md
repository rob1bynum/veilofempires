# v2.7.4

## Agent layout
- Agent page now uses a cleaner two-column dashboard.
- Protocol Loadout moved to a full-width footer at the bottom of the Agent page.

## Protocol synchronization
- Agent Protocol Loadout is rebuilt directly from the current Protocol selections and ranks.
- Removed Protocols no longer remain in the Agent summary.
- Rank changes immediately update the displayed Protocol name.
- Primary and additional repeating Protocol slots are included.
- Duplicate Protocol names are suppressed.

## Preserved
- Identity tab, Contingency placement, Rank 2 progression, XP meter, Equipment catalog, Round Actions, Mission Status, and all existing data fields.

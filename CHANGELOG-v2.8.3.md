# v2.8.3

## Agent page
The dashboard now uses two principal rows:

### Row 1
- Attributes and Defenses
- Skills
- Combat

### Row 2
- Health and Equipment Summary
- Contingencies
- Round Actions and Conditions

Protocol Loadout remains full-width beneath the two rows.

## Fixed
- Agent page content no longer appears on Identity, Protocols, Equipment, Dossier, or Rules.
- Removed the effect of the old `display:block!important` Agent rule through stricter final tab-visibility rules.

## Preserved
- v2.8 color scheme and dashboard styling.
- Protocol synchronization and badges.
- Rank progression, XP meter, Equipment catalog, Round Actions, and all worker automation.
